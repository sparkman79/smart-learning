## 指令
- `/作业` - 提交今日作业（支持多科目）
- `/展开家长版` - 家长查看完整计划（含检查点）

## 身份与权限
- **kid**：默认看孩子版
- **parent/admin**：默认看摘要，可通过 `/展开家长版` 查看家长版

## 数据源
- `data/homework/inbox/` - 原始输入
- `data/homework/parsed/` - 结构化计划
- `data/family/members/index.json` - 身份路由

## 行为

### 1. 接收作业输入
支持两种输入格式：

**常规输入**：
```
/作业
语文：抄写生字 2 遍（20 分钟）
数学：口算 40 题，应用题 3 题（30 分钟）
英语：背单词 10 个（15 分钟）
可用时间：60 分钟
状态：已完成一半
```

**极简输入**：
```
/作业 数学口算 40 题，应用题 3 题，英语背单词 10 个
```

**必须提取字段**：
- 科目
- 数量
- 优先级
- 可用时间
- 当前状态（如果有）

### 2. 拆解任务（硬约束）
| 约束 | 规则 |
|------|------|
| 单任务时长 | **5-20 分钟**（硬约束）|
| 超出 20 分钟 | 继续拆分 |
| 小于 5 分钟 | 合并 |
| 默认排序 | **先易后难** + **先交付后订正** |

### 3. 输出双视图

**🧒 孩子版**（6-10 行，群内直接发送）：
```
1️⃣ 关：数学口算 40 题（15 分钟）
2️⃣ 关：数学应用题 3 题（10 分钟）
⏸️ 休息 5 分钟
3️⃣ 关：英语背单词 10 个（15 分钟）

✅ 完成后回复：已完成
```

**👨👩👧 家长版**（含检查点，需 `/展开家长版` 触发）：
```
📋 今日学习计划
总时长预估：40 分钟

任务顺序：
1. 数学口算 40 题 → 预计 15 分钟 → 完成标准：全部正确
2. 数学应用题 3 题 → 预计 10 分钟 → 完成标准：写出解题步骤
3. 英语背单词 10 个 → 预计 15 分钟 → 完成标准：能拼写 8 个

🔍 检查点（3 个）：
1. 数学是否完成并订正
2. 英语是否完成背诵打卡
3. 全部任务是否收尾
```

### 4. 落盘文件

**inbox（原始输入）**：
```
文件：data/homework/inbox/YYYY-MM-DD_{member_id}_homework.md
```

**parsed（结构化计划）**：
```
文件：data/homework/parsed/YYYY-MM-DD_{member_id}_plan.md
```

## 输出
- **群内**：孩子版或摘要 + "已完成回复：已完成"
- **文件**：inbox + parsed 双文件
- **家长展开**：家长版（含检查点）

## 注意事项
- 文件名必须含日期 + member_id，多孩子不串台
- 任务粒度不稳定，执行层和 timer 接不住
- 群里只发摘要，详细落盘
- 家长版不重复刷长内容
